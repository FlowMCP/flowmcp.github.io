## Public Schemas ✅

**Status: Available**

FlowMCP schemas are publicly available — curated, validated, and open source.
Currently covering transportation, weather, geocoding, and infrastructure data.

- [x] **14 public schemas** across 9 data providers
- [x] **Automated schema catalog** with JSON generation
- [x] **Schema validation** and test infrastructure
- [ ] Expand to additional domains (environment, administration, health)
