## What is FlowMCP?

FlowMCP is an open-source framework that normalizes any data source into AI-callable tools. Built for engineers and decision-makers who want their AI agents to access real-world data without writing brittle integrations.

- **288 data sources** across 13 categories (blockchain, government, weather, etc.)
- **1,534 AI-callable tools** with consistent input/output schemas
- **AI-native:** Designed for LLM-based agents from day one

## <a id="use-cases"></a>Use Cases 

- **Research agents** that pull from open data + APIs without custom code

- **Hackathons & rapid prototypes** with pre-built schemas (see [Hackathon Onboarding](#hackathon))
