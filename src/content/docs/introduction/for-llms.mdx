## What is llms.txt?

The [llms.txt standard](https://github.com/answerdotai/llms-txt) (Jeremy Howard, answer.ai) defines a compact, machine-readable format for project documentation. It allows AI agents to load all relevant context in one step instead of crawling individual pages.
