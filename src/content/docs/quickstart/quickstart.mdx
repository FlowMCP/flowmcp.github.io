## Install the CLI

The FlowMCP CLI is the fastest path to a working setup. It lets you search the {toolCount}-tool catalog, add schemas, and call APIs without writing code.

<InstallNote repo="flowmcp-cli" global />

Verify the install:

```bash
flowmcp --version
flowmcp list
```

`flowmcp list` shows every tool from the configured `schemaFolders[]`, with a `disabledCount` for any tools missing an API key. Continue below to make your first call.

## Call Your First Schema (CLI mode)

```bash
flowmcp search coingecko
flowmcp call coingecko-tokens.simplePrice '{"ids":"bitcoin","vs_currencies":"usd"}'
```

Two commands and you have a live API call — there is no activation step. The CLI is the recommended path for hackathons, quick prototypes, and exploring the catalog.

## Prerequisites

- **Node.js 22+** — check with `node --version`
- **npm** — comes with Node.js

:::note[Embedding FlowMCP in your own code?]
The CLI is the fastest path for setup, hackathons, and exploring the catalog. If you want to embed FlowMCP directly in a Node.js application — load schemas, validate them, and call APIs programmatically — use the core library instead.
:::
