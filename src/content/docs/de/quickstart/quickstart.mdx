## CLI installieren

Die FlowMCP CLI ist der schnellste Weg zu einem funktionierenden Setup. Sie laesst dich den Katalog mit {toolCount} Tools durchsuchen, Schemas hinzufuegen und APIs aufrufen — ohne Code zu schreiben.

<InstallNote repo="flowmcp-cli" global />

Installation pruefen:

```bash
flowmcp --version
flowmcp list
```

`flowmcp list` zeigt jedes Tool aus den konfigurierten `schemaFolders[]`, mit einem `disabledCount` fuer Tools, denen ein API-Schluessel fehlt. Weiter unten taetigst du deinen ersten Aufruf.

## Erstes Schema aufrufen (CLI-Modus)

```bash
flowmcp search coingecko
flowmcp call coingecko-tokens.simplePrice '{"ids":"bitcoin","vs_currencies":"usd"}'
```

Zwei Befehle und du hast einen Live-API-Aufruf — es gibt keinen Aktivierungsschritt. Die CLI ist der empfohlene Weg fuer Hackathons, schnelle Prototypen und das Erkunden des Katalogs.

## Voraussetzungen

- **Node.js 22+** — pruefen mit `node --version`
- **npm** — wird mit Node.js ausgeliefert

:::note[FlowMCP in eigenen Code einbetten?]
Die CLI ist der schnellste Weg fuer Setup, Hackathons und das Erkunden des Katalogs. Wenn du FlowMCP direkt in eine Node.js-Anwendung einbetten willst — Schemas laden, validieren und APIs programmatisch aufrufen — nutze stattdessen die Core Library.
:::
