## Was ist FlowMCP?

FlowMCP ist ein Open-Source-Framework, das beliebige Datenquellen in KI-aufrufbare Tools normalisiert. Gebaut fuer Engineers und Entscheider, deren KI-Agenten ohne fragile Integrationen auf reale Daten zugreifen sollen.

- **288 Datenquellen** in 13 Kategorien (Blockchain, Behoerden, Wetter, etc.)
- **1.534 KI-aufrufbare Tools** mit konsistenten Input/Output-Schemas
- **KI-nativ:** Von Tag eins fuer LLM-basierte Agenten konzipiert

[Zum Schnellstart →](/de/quickstart/quickstart/)

## <a id="trust"></a>Vertrauen & Open Source 

- Lizenz: **MIT** — nutzen, forken, verteilen
- GitHub: [github.com/FlowMCP](https://github.com/FlowMCP) — aktive Repositories
- Test-Coverage: pro Repo publiziert (siehe [Repository-Uebersicht](https://github.com/FlowMCP))
- Kein Vendor Lock-in: Schemas, Core und CLI sind unabhaengig und offline lauffaehig
